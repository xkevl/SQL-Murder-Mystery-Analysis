# SQL Murder Mystery Analysis

## Project Overview
This project involves solving a fictional murder case using SQL by analyzing relational database tables.

## Objective
Identify the suspect by querying multiple tables and following clues from the dataset.

## Skills Used
- SQL (SELECT, WHERE, JOIN)
- Data filtering and analysis
- Problem-solving with relational data

## Approach
1. Retrieved crime scene report using filters
2. Identified witnesses based on location clues
3. Cross-referenced suspects using JOINs and filtering
4. Narrowed down results using logical conditions

## Key Queries
See `queries.sql` for full solution.

## Final Result
Identified the suspect by combining multiple clues across different tables.

## What I Learned
- How to navigate relational databases
- Writing efficient SQL queries
- Breaking down complex problems into steps
